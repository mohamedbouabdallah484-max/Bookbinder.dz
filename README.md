# Bookbinder.dz v2
### أداة حرفيي تجليد الكتب — Book Binding Professional Tool

---

## 📖 عن البرنامج

**Bookbinder.dz** هو تطبيق احترافي مخصص لحرفيي تجليد الكتب، يعمل على نظام Windows بدون الحاجة لتثبيت أي برامج إضافية.

---

## ✨ المميزات

### 🩺 Content Doctor — طبيب المحتوى
- تنظيف خلفية الصفحات
- تحسين الحدة والوضوح
- تقسيم الصفحات المزدوجة
- تدوير الصفحات
- توحيد قياس الصفحات
- إضافة هامش
- علامة مائية
- تلوين الخلفية

### 📐 The Architect — الفرض الطباعي
- فرض ثنائي 2-up (كتاب A5 من ورق A4)
- فرض رباعي 4-up (كتاب A6 من ورق A4)
- دعم الكتب العربية RTL والفرنسية LTR
- خيار الكتاب كاملاً ملزمة واحدة
- علامات القص (Crop Marks)
- علامات الكعب (Collating Marks)

### 🎨 The Artist — الفنان
- ترقيم الصفحات
- ضغط الملف
- إصلاح PDF
- تشفير وفك تشفير

### 📁 Manager — المدير
- دمج الملفات
- استخراج الصفحات
- حذف الصفحات
- تحويل PDF إلى صور JPG/PNG
- تحويل صور إلى PDF
- معلومات الملف

---

## 💻 متطلبات التشغيل

- Windows 10 / 11 (64-bit)
- لا يحتاج تثبيت Python
- لا يحتاج اتصال بالإنترنت
