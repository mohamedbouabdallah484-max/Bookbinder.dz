# Bookbinder.dz v2
### أداة احترافية لحرفيي تجليد الكتب

---

## 📖 عن البرنامج

**Bookbinder.dz** تطبيق متخصص لحرفيي تجليد الكتب، يجمع كل أدوات تحضير ملفات PDF للطباعة والتجليد في مكان واحد. يعمل على Windows بدون تثبيت أي برامج إضافية.

---

## 🗂 أقسام البرنامج

### 🏠 الصفحة الرئيسية
- 🔍 فحص حالة النظام (RAM / CPU / ملفات مؤقتة)
- 🧠 تحرير الذاكرة
- 🗑 حذف الملفات المؤقتة
- 🎧 قسم الدعم الفني والتواصل

---

### 📐 هندسة الكتاب
*التحضير الهندسي للصفحات قبل التجليد*

| الأداة | الوصف |
|--------|--------|
| ✂️ تقسيم الصفحات المزدوجة | Split Spread — عربي أو فرنسي |
| 🔄 تدوير الصفحات | كل الصفحات أو صفحة محددة |
| 📐 توحيد القياس | A4 / A5 / B5 / A3 |
| 📏 إضافة هامش Gutter | بدون فقدان جودة |
| ✂️ قص الهوامش | Auto-Crop — إزالة الفراغ الزائد |
| 🎨 تلوين الخلفية | إضافة لون شفاف للصفحات |

---

### 🩺 علاج المحتوى
*تجهيز جودة الطباعة ومحتوى الصفحة*

| الأداة | الوصف |
|--------|--------|
| 🧹 تنظيف الخلفية | إزالة الاصفرار من الكتب المصورة |
| ✨ تحسين حدة النص | للكتب المصورة منخفضة الجودة |
| 🔢 ترقيم الصفحات | للكتب غير المرقمة |
| 🗜 ضغط الملف | للكتب الكبيرة الحجم |
| 🔧 إصلاح PDF التالف | إعادة بناء الملف التالف |
| 🔓 فك كلمة السر | إزالة الحماية بكلمة السر |
| ⬛ تحسين السواد | Black Re-assertion |

---

### 🏗 هندسة الملازم
*تجهيز الكتاب للطباعة الاحترافية والخياطة*

| الأداة | الوصف |
|--------|--------|
| ✂️ تقسيم إلى ملازم | الملزمة المعيارية 32 صفحة |
| 📋 الفرض الطباعي للملازم | 2up و 4up — RTL و LTR — ملزمة واحدة كاملة |
| 📄 إدراج صفحات بيضاء | بطانة أو فاصل |
| 🖨 تجهيز الطباعة المزدوجة | للطابعات أحادية الوجه |
| 📌 علامات الكعب | Collating Marks — مربع / دائرة / نقطة |
| ✂️ علامات القص | Crop Marks للطباعة الاحترافية |

---

### 📁 إدارة الملفات
*تنظيم وإدارة ملفات PDF باحترافية*

| الأداة | الوصف |
|--------|--------|
| 🔗 دمج الملفات (متقدم) | ترتيب وحذف وتعديل |
| 📁 دمج مجلد كامل | دمج تلقائي بالترتيب الأبجدي |
| ✂️ استخراج صفحات | نطاقات: 1-5, 8, 12 |
| 🗑 حذف صفحات | حذف بالرقم أو النطاق |
| 🖼 تحويل PDF إلى صور | JPG/PNG — كل الكتاب أو صفحات محددة |
| 🖼 تحويل صور إلى PDF | JPG/PNG → PDF مرتبة أبجدياً |
| ℹ️ معلومات الملف | صفحات، حجم، أبعاد |
