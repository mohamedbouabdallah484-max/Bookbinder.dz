# 📚 Bookbinder.dz — v3

**أداة احترافية لتجليد الكتب وتحرير ملفات PDF**

---

## 📥 تحميل

👇 اضغط على **Releases** في الجانب الأيمن لتحميل البرنامج

> متطلبات التشغيل: Windows 10/11 — لا يحتاج Python أو أي تثبيت إضافي

---

## ✨ المميزات

### 🛠 أدوات سريعة
- 🔗 دمج ملفات PDF
- ✂ تقسيم الملف
- 🗜 ضغط الملف
- 🔓 فك كلمة السر
- 🖨 تجهيز وجهين للطباعة
- 🔧 صيانة وإصلاح الملفات التالفة
- 🖼 تحويل PDF إلى صور مع دعم نطاق الصفحات
- 📄 تجميع صور في PDF
- 📚 Smart Booklet — ملازم جاهزة للطباعة
- ⚡ معالجة دفعية لعشرات الملفات

### 📐 هندسة الكتاب
- ✂ تقسيم الصفحات المزدوجة (عربي / فرنسي)
- 🔄 تدوير الصفحات (90 / 180 / 270 + حر)
- 📏 توحيد القياس (A3، A4، A5، B4، B5)
- ➕ إضافة هامش للتجليد
- ✂ قص الهوامش الزائدة

### 💊 علاج المحتوى
- 🧹 تنظيف الخلفية وإزالة الاصفرار
- 🔍 تحسين حدة النص حتى 8×
- 🖊 تحسين سواد النص حتى 5×
- 🎨 تلوين الخلفية بخوارزمية ذكية تحافظ على النص
- 💧 إضافة علامة مائية

### 📋 إدارة الصفحات
- 📋 استخراج / حذف / إدراج صفحات
- 🔢 ترقيم بأرقام عربية ١٢٣ أو لاتينية 123
- 🔃 عكس ترتيب الصفحات

### 📚 هندسة الملازم
- 📖 فرض ثنائي 2-up
- 📚 فرض رباعي 4-up
- ✂ علامات القص
- 📌 علامات الكعب

### 🚀 المحرر السريع
معالج ذكي يجمع كل العمليات في نافذة واحدة

---

## ⌨️ اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl+O` | فتح ملف PDF |
| `Ctrl+S` | حفظ الملف |
| `Ctrl+Z` | تراجع |
| `Ctrl+W` | إغلاق الملف |
| `Ctrl+M` | دمج ملفات |
| `Ctrl+B` | Smart Booklet |
| `← →` | التنقل بين الصفحات |
| `Ctrl++/-` | تكبير / تصغير |
| `F1` | المساعدة |
